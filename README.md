A lightweight and modular mod that works well with most other mods. Built around how I wanted loot and gear value tracking to work.


![Preview](https://i.imgur.com/Jl8iCNE.png)

It includes a built in price display to support accurate equipment value, loot value, and end of raid calculations.
If you prefer another pricing mod or simply don’t want the built in price display, you can disable it in the F12 config.

![Raid loot value](https://i.imgur.com/vZaTbAA.png)

You can choose between trader sell prices or flea prices, and most features can be enabled, disabled, or adjusted depending on how you want the mod to behave.

The goal isn’t to replace existing pricing mods, but to provide accurate and reliable value calculations both during and outside of raids.

### Features

- Shows item values on hover
- Shows value of loot in backpacks, rigs and cases
- Shows total equipment value outside raids
- System marks eqiupment so it wont affect loot value
- Tracks and displays loot value during raids
- Simple display with total loot value at the end of a raid
- Cool money animations

### Configuration (F12 menu)

- Enable or disable tooltip prices
- Switch between trader and flea prices
- Toggle breakdowns such as ammo, contents, plates, and attachments
- Choose between compact or detailed display




![Compatibility](https://i.imgur.com/R7dpGR4.png)

### Compatibility

Built to work with most mods, but compatibility can vary with mods that replace tooltip logic or heavily change trader or flea pricing.

### Tested with

- swiftxp ShowMeTheMoney 2.7.0
- Tyfon UI Fixes 5.3.8
- DrakiaXYZ Live Flea Prices
- SVM price multipliers
